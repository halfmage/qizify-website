# Target-account list: how to build it + worked examples

## Sources to build the list (DACH, Germany-first)
- **mein-now.de** (Bundesagentur für Arbeit provider portal, 20k+ providers): filter by AZAV + Umschulung/Reha + region + size. Primary discovery engine.
- **Arbeitsgemeinschaft Deutscher Berufsförderungswerke**, ~24-28 BFW, finite high-value Tier 1 reha list.
- **BBB (Bundesverband der Träger beruflicher Bildung)**, ~170 members; core VET providers. Also a channel/credibility play.
- **Wuppertaler Kreis**, ~50 commercial leaders (~€1.5B combined revenue); well-funded.
- **Private university rankings**, IU, FOM, SRH and the top 5 (≈54% of private students).
- **AZAV fachkundige Stellen** (CERTQUA, TÜV, DEKRA, GUTcert), not targets, but referral partners upstream of every compliance decision.

## Scoring (rank each account before outreach)
Score 1-3 on each, prioritize totals ≥ 10:
1. **Need trigger**, under BFSG/AZAV/SGB IX pressure, first-time digitization, or AI initiative?
2. **Scale**, multi-site / large cohorts (justifies a build, spreads per-seat cost)?
3. **Funding path**, corporate/tuition capex, or self-funding foundation, or ESF+ candidate?
4. **Digital signal**, has commissioned external software / has an E-Learning role / weak incumbent platform?

## Worked examples

### Teutloff (Technische Akademie), Tier 1, score ~11
- Multi-site (Braunschweig HQ + 5-6 locations), 250+ trainers, IHK Umschulung/Weiterbildung + corporate in-house.
- Signal: already runs an e-learning portal commissioned from an external IT vendor → proven custom-software willingness.
- Angle: extend/upgrade the existing portal with an AI learning coach + AZAV-supporting documentation; per-seat opex.
- Committee: two gGmbH Geschäftsführer (economic); influence via REFA Nordwest / BIT GmbH owners.

### A large BFW (reha), Tier 1 archetype, score ~11
- Institutional, DRV-funded foundation, ~100 locations across the BFW network; can self-fund infrastructure.
- Need: SGB IX accessibility + adaptive paths for rehab learners + BFSG.
- Angle: accessible, individualized AI learning platform; lead with BFSG/WCAG and learner outcomes.
- (Use Lavie-Reha only as the messaging archetype; it is too small to be a primary target.)

### A private university (IU / FOM / SRH), Tier 1, score ~10
- Scaling fast, digital-native, has IT + existing platform.
- Angle: custom AI tutor/chatbot layer + integrations, not full LMS replacement; capex, faster cycle.

## Cadence
Build a rolling list of ~40-60 scored accounts; work the top 15-20 via the playbook. Re-score on new triggers (tender wins, AZAV recertification windows, funding deadlines).
